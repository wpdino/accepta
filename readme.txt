== Description ==

Accepta is a modern, clean, and highly customizable WordPress theme designed to work seamlessly with the Elementor page builder. Built by WPDINO, this theme provides a solid foundation for creating professional websites with ease.

**Key Features:**
* **Elementor Compatible** - Fully optimized for Elementor page builder
* **Customizer Options** - Extensive customization options through WordPress Customizer
* **Responsive Design** - Mobile-first approach ensuring your site looks great on all devices
* **Typography Control** - Advanced typography controls with Google Fonts integration
* **Layout Options** - Flexible layout options including sidebar positions and container width
* **Sticky Header** - Optional sticky header functionality
* **Footer Customization** - Customizable footer with widget areas and social media integration
* **Modern Design** - Clean, minimalist design with attention to detail
* **Performance Optimized** - Lightweight and fast-loading theme

Perfect for business websites, portfolios, blogs, and any type of website that requires flexibility and professional appearance.

== Installation ==

1. Upload the theme files to `/wp-content/themes/accepta` directory, or install the theme through the WordPress themes screen directly.
2. Activate the theme through the 'Themes' screen in WordPress.
3. Navigate to Appearance > Customize to configure the theme settings.
4. Install and activate Elementor page builder for the best experience (optional but recommended).

== Frequently Asked Questions ==

= Is Accepta compatible with Elementor? =
Yes! Accepta is fully compatible with Elementor page builder and is optimized to work seamlessly with it.

= Can I customize the theme colors? =
Yes, you can customize colors through the WordPress Customizer under the Colors section.

= Does Accepta support custom logos? =
Yes, Accepta supports custom logos through the WordPress Customizer under Site Identity.

= Can I change the layout of the site? =
Yes, you can customize the container width, sidebar position, and content spacing through the Layout section in the Customizer.

= Does Accepta support sticky header? =
Yes, Accepta includes an optional sticky header feature that can be enabled or disabled through the Customizer under Header section.

= Is Accepta translation ready? =
Yes, Accepta is translation ready and includes .pot file for translations.

= What widgets are included? =
Accepta includes standard WordPress widgets and is compatible with all WordPress widgets and plugins.
